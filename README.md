jlunar
======

jcalendar.h jcalendar.cpp        农历月历位置计算和绘图示范应用

pylunar.cpp                      农历计算的python扩展

setup.py                         python扩展安装脚本：python setup.py install

junar.js                         JavaScript农历计算

getlunar.cpp                     获取javascript函数需要的农历数据
